# Telegram Birthday Bot (Python)

Телеграм-бот на Python для хранения дней рождения друзей и напоминаний о приближении события.

## Возможности
- Добавление/обновление дня рождения: `/add имя дата`
- Удаление по имени: `/delete имя`
- Просмотр списка: `/list`
- Подсказка по командам: `/help`
- Ежедневная проверка и напоминания за N дней до события (по умолчанию за 3 дня)

## Форматы даты
Поддерживаются:
- `DD.MM` (например `14.02`)
- `DD.MM.YYYY` (например `14.02.2004`)
- `YYYY-MM-DD` (например `2004-02-14`)
Также можно использовать `-` или `/` как разделители.

## Хранение данных
Данные сохраняются в SQLite базе `birthdays.sqlite3`.
У каждого пользователя Telegram — свой список дней рождения (по `user_id`).

## Требования
- Python 3.10+
- macOS/Linux (для Windows шаги аналогичные)
